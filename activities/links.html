What is the root? What is the relationship between folder and file names and the path?

Root is where all the folders locate. When you create a webpage, it is located in a folder in your root. The path tells you the name of the file for the webpage, and the name of the folder containing the file.

What is a default document?

If you go to the root of the address, you will only see the list of folders. If you want to have an actual webpage for the root, you can add file called a default document in your root and design it.

Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link has the full address, and a relative link has only the part of the address from which the addresses starts to differ. I would use relative links because it is shorter and convenient.
